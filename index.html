    How to deploy on Vercel (easiest)

    1. Click the Deploy on Vercel button below
    2. Sign in with your GitHub account
    3. Click Deploy — that's it! Your proxy URL will be auto-detected

    How to set up Cloudflare Workers (optional)

    Cloudflare Workers is a free alternative backend. It can be faster in some regions and supports larger file downloads.

    1. Install Node.js if you don't have it:
      Download from nodejs.org and install it
    2. Clone the repo — open your Terminal (Mac) or Command Prompt (Windows) and run:
      git clone https://github.com/amirhp-com/google-mirror.git
    3. Go into the folder:
      cd google-mirror
    4. Log in to Cloudflare:
      npx wrangler login
      This opens your browser — sign in or create a free Cloudflare account
    5. Deploy the worker:
      npx wrangler deploy worker/proxy-worker.js --name webgate-proxy
    6. Copy your Worker URL — it will look like:
      https://webgate-proxy.your-name.workers.dev
    7. Paste it in the "Cloudflare Worker URL" field above, and check "Use Cloudflare Worker as proxy backend"
